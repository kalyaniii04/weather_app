## Description

The **Weather Forecasting Web Application** is a responsive web app that provides real-time weather information for cities worldwide using the OpenWeather API. It focuses on API integration, asynchronous JavaScript, and dynamic UI updates.

This project was built to practice working with third-party APIs and modern frontend development concepts.

---

## Features

- Real-time weather data for 200,000+ cities
- OpenWeather REST API integration
- Asynchronous data fetching using Fetch API
- Dynamic DOM updates for live weather details
- Deployed on Render

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **API:** OpenWeather API
- **Deployment:** Render

---

## Run Locally

```bash
git clone https://github.com/kalyaniii04/weather_app.git
cd weather_app
npm install
npm start
```

App runs at: `http://localhost:3000`
